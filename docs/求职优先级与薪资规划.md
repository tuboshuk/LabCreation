   - 飞书开放平台 + 大模型应用（AI 应用工程）

## 📚 附录：融资轮次科普

### 创业公司融资阶段

```
种子轮/天使轮 → A 轮 → B 轮 → C 轮 → D 轮及以后 → IPO 上市
    ↓           ↓      ↓      ↓         ↓          ↓
  刚起步     产品上线  模式验证  快速扩张  成熟期     上市
  1-10 人     10-50 人  50-150 人 150-500 人 500 人 +   数千人
```

### 为什么推荐 B 轮-C 轮？

```
✅ 优势：
  1. 风险适中（基本能活 2-3 年）
  2. 薪资中上（20-35k）
  3. 期权有价值（可能变现）
  4. 技术栈新（愿意用新技术）
  5. 成长快（一人多职，接触面广）
  6. 工作强度比大厂低（有时间副业）
  7. 面试难度比大厂低（更看重实战）

❌ 劣势：
  1. 还是有倒闭风险（但概率低）
  2. 流程可能不规范（需要你自己建立）
  3. 可能需要加班（但比大厂 996 好）
```
