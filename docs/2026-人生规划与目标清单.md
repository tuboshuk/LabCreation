## 3. 假期使用策略

### 3.1 年假规划

```
当前公司（离职时）：
  - 未休年假：折算工资
  - 5 天 × (12k ÷ 21.75) = 2,758 元
  - 离职时一起结算

  - 年假：5 天（按社会工龄）
  - 有些公司给 7-10 天（福利年假）
  - 年假价值：28k ÷ 21.75 × 5 = 6,437 元

使用计划：
  5 月订婚：2 天（5/2、5/5）
  剩余 3 天：12 月休（错峰旅游）

```
北京婚假：10 天（3 天法定 +7 天奖励）

使用计划：
  10/1-10/7：国庆假期（7 天）
  10/8-10/17：婚假（10 天）
  总计：17 天连续假期！

  - 入职后 1 个月（5 月中旬）
  - 提前告知 HR 和领导
  - 显得规划清晰

### 3.3 假期总价值

```

    - 延长面试周期（到 5 月）

  概率：30%
  应对：

  概率：40%
  应对：
    - 动用副业收入
    - 双方家庭支持
    - 简化部分环节

风险 4：新工作太忙（没时间副业）
  概率：50%
  应对：
    - 调整副业模式（外包为主）
    - 降低副业预期（月入 3000 即可）

  应对：
    - 优先保证睡眠（7-8 小时）

## 8. 核心原则

### 8.1 做事方法

```
✅ 一次只推进一个主线
  3-4 月：专注跳槽
  6-12 月：跳槽 + 副业
  5 月 +10 月：生活优先

✅ 先卖再做
  优先用 99 元诊断验证需求
  再做定制开发和标准化产品

✅ 先做资产再做产品
  先沉淀案例文档、技术方案、SOP
  再做功能开发

✅ 以可持续为上
  不熬夜，不透支
  保证睡眠 7-8 小时
  每周运动 3 次
```

```
3-4 月：
  主业 70% + 副业 20% + 生活 10%

  主业 50% + 订婚 40% + 副业 10%

  主业 50% + 副业 40% + 生活 10%

  主业 50% + 婚礼 40% + 副业 10%

  主业 30% + 婚礼 60% + 副业 10%

  主业 50% + 副业 40% + 生活 10%
